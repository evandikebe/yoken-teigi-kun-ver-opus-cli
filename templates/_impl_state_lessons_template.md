# 走行内レッスン (lessons.md) — 案件内エピソード記憶 (Reflexion)

> このファイルは **同一案件の実装走行中だけ効く速い学習ループ**です（IMPL_RULES R-9.4 / §4.2）。
> 各 impl-engineer は **起動時にこのファイルを Read** し、検証ループ(R-9)で「失敗 → 原因特定 → 回復」したら **1行追記**します。
> 同一走行中の他チケットが同じ轍を踏まないための共有メモであり、retrospective が回す「案件をまたぐ遅い学習ループ」とは別物です。

## 運用ルール

- **粒度**: 1件1行。「どのチケットで・何が起きて・原因は何で・どう回避するか」を端的に。
- **上限**: 直近 **20件** 程度（コンテキスト腐敗の防止）。超えたら古い項目から「恒久ルール候補」へ移すか削る。
- **昇格**: 複数チケットで再発した教訓は、案件完了時に **retrospective が拾い、IMPL_RULES への恒久ルール化を提案**する（採否は人間ゲート）。
- **書かないもの**: 案件固有でない一般論、シークレット・PII、愚痴。事実と回避策だけ。
