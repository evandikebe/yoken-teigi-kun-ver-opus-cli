# 未解決事項チケット

このファイルは、ヒアリング中・設計中に出てきた「まだ答えが出ていない論点」を一覧化します。
各項目は **解消するか、「実装中に決める」と明示する** までクローズしないこと。

## 形式

```markdown
### OQ-001: <論点タイトル>
- 出元: <Phase X / どのドキュメント>
- 状態: open | resolved | deferred (実装中決定)
- 関連要件: <F-001 / NF-性能-02 等>
- 内容: <論点の背景>
- 想定の選択肢:
  - 案A: ...
  - 案B: ...
- 担当: <ユーザー / 設計者 / 実装者>
- 期限: <YYYY-MM-DD or マイルストーン>
```
