# Q&A ログ

ヒアリング中にチャットで番号付きに投げた質問と、ユーザーが選んだ回答を時系列で追記する。
**このログは設計判断の根拠**になるので削除しないこと。回答が変わったら新エントリを追加し、旧エントリには `〔更新済み: → entry-XXX〕` と注記。

---

## 形式

```markdown
### entry-001  YYYY-MM-DD HH:MM  Phase 1: 要件定義

**質問1**: 想定する利用規模はどれくらいですか？
- 選択肢: (a) 〜10人 / (b) 〜100人 / (c) 〜1万人 / (d) それ以上
- **回答**: (b) 〜100人
- メモ: 社内利用、最終的にはグループ全社200名規模に拡大想定

**質問2**: いつまでに動くものが欲しいですか？
- 選択肢: (a) PoC 1-2週 / (b) 3ヶ月以内 / (c) 半年以上 / (d) 未定
- **回答**: (b) 3ヶ月以内
```
